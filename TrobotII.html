5  Thermocycling

5.1 장비소개

  • 로봇암과 연계된 thermal cycle

  • 96 well block x 4 (0.2 ml tubes, 96 well microplates, 8 well strips)

  • Silver block with gold coating

  • Proposed sample volume: 5- 100 uL

  • Maximum heating: 5.0 ℃/s

  • Average heating: 4.9 ℃/s

5.4 사용법

5.4.1 사용 시 주의사항

  • PCR 실험을 할 때, 4℃, ∞ 로 store 기능을 사용하는 경우가 많은데 ’STOP’을 누르지 않고 ’OPEN’만 하여 plate/tube를 회수하게 되면 온도차에 의해 block에 물이 고이게됨.

5.4.2 PCR tube

  • 0.2 ml PCR 8-Tube Strips (Bioneer, Cat. No. T1C-028-N): dome type cap, tube와 cap 결합 시 높이 23 mm로 사용 가능

  • Tube를 사용한 적은 샘플 이용 시 뚜껑을 닫는 압력에 의해 tube가 찌그러질 수 있고 장비에 무리를 줄 수 있음. 아래 그림처럼 압력을 고르게 받을 수 있도록 배치하여 사용.

5.4.3 PCR plate

  • 96 well PCR plate (Bio-Rad, Cat. No. HSP9601, MRO): low-profile 로 높이 16 mm, 사용 가능

  • twin.tec PCR plate 96, skirted (Eppendorf, Cat. No. 0030128648), 사용 가능

5.4.8 Protocol setting

  • Table mode

  • GRAPH mode

  • Edit

    • Step type: 자주 쓰는 양식에 대한 예시 선택 가능, blank 선택 후 add해서 설정 가능함.

    • Temperature : block 설정온도

    • ΔT: touch town PCR 과 같이 cycle이 증가할 때마다 온도 조건을 바꿔서 실험할 때 사용하는 값

    • Hold time: Temperature에서 설정한 온도에 도달 후 지속되는 시간

    • Δt: cycle이 증가할 때마다 hold time이 길어지게 설정할 수 있음

    • ΔR: ramping rate, 다음 스텝의 온도까지 변화할 때의 속도를 조정할 수 있음.

    • Goto: 반복하고자하는 스텝이 있을 경우 사용

    • Clycles: 반복하는 횟수

  • Login:: Homescreen

5.4.9 Run

  • 장비 선택 - 프로그램 선택 - START - CLOSE LID

  • 반응 진행 시 block 색깔이 주황색으로 바뀜

  • Temp. 4℃: block 표시가 파란색으로 바뀜

  • OPEN

  • STOP: PCR 종료 후 ’STOP’을 꼭 눌러주세요

  • 프로그램이 종료된 후에는 MONITOR 창의 BLOCK 가 활성화되지 않음.
